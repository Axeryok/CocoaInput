CocoaInput
***
Support IME input(Japanese,Chinese,Korean) in Minecraft on Mac OS X.

###Compiling
***
```Bash
git clone https://github.com/Axeryok/CocoaInput.git
cd CocoaInput
gradle build
```
Jars will be located in "build/libs".

###
###Installing
***
